# Code of Conduct

This project follows a simple contributor standard:

- Be respectful and direct.
- Assume good intent, but accept correction quickly.
- Keep critique focused on code, docs, design, and user safety.
- Do not harass, threaten, demean, or publish private information.
- Be especially careful with security reports and vault-related data.

Maintainers may remove comments, issues, pull requests, or contributors that
make the project unsafe or unwelcoming.
